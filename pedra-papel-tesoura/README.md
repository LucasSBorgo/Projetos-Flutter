# Projeto: Pedra x Papel x Tesoura

Nesse projeto é possível escolher entre 3 opções: Pedra, Papel ou Tesoura.
Ao selecionar uma das 3, também será escolhido uma das 3 opções que será seu adversário. E, seguindo as regras do jogo tradicional será gerado um texto informando se sua escolha ganhou, perdeu ou empatou.
